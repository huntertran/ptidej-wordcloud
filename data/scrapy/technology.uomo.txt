The scope of the Eclipse UOMo project is to host a variety of implementation activities related to UCUM or other unit-specific standards like UnitsML or MathML:
Unit Standards
Basic implementation of
UCUM
Logic for use in any context
Help to incorporate units into
W3C
standard
MathML
Support
OASIS
standard
Units Markup Language (UnitsML)
Implementing Units of Measurement API
Strong Unit Typing (Compile time support)

JSR 363
.

https://github.com/unitsofmeasurement/unit-tck-usage/tree/uomo

The minimum JRE version for UOMo 0.7 is Java 7.
